Astra – AI-Powered Penetration Testing Assistant

> Empowering Ethical Hackers with AI

Astra is an advanced, open-source tool that revolutionizes penetration testing by integrating state-of-the-art AI models into your command-line interface. Designed for cybersecurity professionals, ethical hackers, and developers, Astra automates vulnerability assessments and enhances security workflows with real-time insights and automated note-taking.

Combining AI-driven testing with open-source cybersecurity tools, Astra empowers teams to perform comprehensive vulnerability assessments and uncover hidden threats faster than ever before.

Installation

System Requirements:

  • At least 8GB of GPU memory (tested with 12GB)
  • Python 3.11 or higher

Installation Command:

python -m pip install astra-ai --upgrade

Running Astra

Important: On your first run, you’ll be prompted to select a cache directory where Astra will download your chosen AI model. Follow these steps:

  1. Create a free Hugging Face Account and generate an access token.
  2. Export your token to the CLI:
    export HF_TOKEN=YourTokenHere
  3. Launch Astra and monitor the download progress:
    astra

Interacting with the Models

Begin your input with a ! (e.g., ! write a python script to scan open ports).

Key Features

  • AI-Powered Internet Search: Real-time cybersecurity insights from the web.
  • AI-Assisted Note-Taking: Automatically record and categorize security findings.
  • Real-Time AI-Driven Insights: Get suggestions for exploiting vulnerabilities.
  • Enhanced Tool Integration: Import data from external tools seamlessly.
